# taubyte/utils

This repository contains golang utilities for doing things such as:
- Caching a map in memory with a TTL
- Generating UUIDs
- Filesystem utilities
- Hex encoding/decoding and utilities
- uuid utilities
- map display and utilities
- path utilities
